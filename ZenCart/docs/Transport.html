条件

未分包，则显示运输方式

计算公式

1）IF  重量 <= 首重，THEN  基本运费 = 首重邮费
2）IF 重量 > 首重，THEN 基本运费 = 首重邮费 + 向上取整((包裹重量 C 首重 ) / 续重) * 续重金额
3)  IF 是偏远地区, THEN 基本运费 = 基本运费  + 偏远地区附加费
4)  运费 = (基本运费 + 附加费) * 折扣
